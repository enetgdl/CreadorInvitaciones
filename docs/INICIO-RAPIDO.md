# 🚀 INICIO RÁPIDO - Sistema de Invitaciones Digitales

## ⚡ Pasos para Comenzar (5 minutos)

### 1️⃣ Abrir el Editor
```
1. Asegúrate que XAMPP esté corriendo
2. Abre tu navegador
3. Ve a: http://localhost/Invitacion/
```

### 2️⃣ Ver Ejemplo Funcionando
```
Abre: http://localhost/Invitacion/ejemplo.html
Verás una invitación de XV años de ejemplo
```

### 3️⃣ Crear tu Primera Invitación

#### Tab GENERAL (Datos básicos)
```
✅ Tipo de evento → XV Años / Boda / Bautizo
✅ Nombre del evento → "Mis XV Años"
✅ Nombre del festejado → "María González"
✅ Fecha → 15/03/2026
✅ Hora → 19:00
✅ Lugar → "Salón Los Pinos"
✅ Dirección → "Av. Principal #123"
```

#### Tab DISEÑO (Apariencia)
```
🎨 Tema de color → Click en preset dorado
🎨 O personaliza:
   - Color principal → #FFD700
   - Color secundario → #FFF8DC
   - Fuente título → Great Vibes
   - Fuente texto → Montserrat
🎨 Efecto fondo → Partículas
```

#### Tab CONTENIDO (Mensajes)
```
✍️ Mensaje bienvenida → "Con mucho amor te invitamos..."
✍️ Mensaje principal → "Quiero compartir contigo..."
✍️ Código vestimenta → "Formal Elegante"
✍️ Hashtag → "#MisXV2026"
```

#### Tab MULTIMEDIA (Opcional)
```
📸 Foto del festejado → Click "Elegir archivo"
🎵 Música de fondo → MP3 de tu elección
🎥 Video de fondo → (Opcional)
```

#### Tab INTERACTIVO
```
⏰ Cuenta regresiva → ✅ Activar
🔲 Código QR → URL de confirmación
🗺️ Mapa → Coordenadas del lugar
```

### 4️⃣ Vista Previa
```
La vista previa se actualiza automáticamente
Verifica que todo se vea bien en el panel derecho
```

### 5️⃣ Exportar
```
1. Click en botón "Exportar" (arriba derecha)
2. Selecciona opciones:
   ✅ Minificar código
   ✅ Optimizar imágenes
3. Click "Descargar ZIP"
4. Se descarga archivo invitacion.html
```

### 6️⃣ Publicar
```
Opción más fácil:
1. Ve a https://app.netlify.com/drop
2. Arrastra tu archivo invitacion.html
3. ¡Listo! Copia el link y comparte
```

## 🎯 Funciones Principales

### Guardado Automático
```
✅ Se guarda cada 5 segundos automáticamente
✅ También puedes guardar manualmente con botón "Guardar"
✅ Los datos se mantienen en tu navegador
```

### Cuenta Regresiva en Tiempo Real
```
✅ Se calcula desde la fecha/hora actual
✅ Muestra: Días, Horas, Minutos, Segundos
✅ Se actualiza cada segundo
✅ Al llegar a cero muestra "¡El evento ha comenzado!"
```

### Códigos QR Automáticos
```
✅ Se generan automáticamente
✅ Puedes usar:
   - Link de Google Maps
   - Link de formulario de confirmación
   - Cualquier URL
```

### Mapas Integrados
```
Para obtener coordenadas:
1. Abre Google Maps
2. Click derecho en la ubicación
3. Copia los números (ejemplo: 19.4326, -99.1332)
4. Pega en "Coordenadas del lugar"
```

## 🔧 Solución Rápida de Problemas

### No se ve la vista previa
```
→ Click en botón 🔄 Refrescar
→ Espera 2 segundos después de hacer cambios
```

### No puedo subir archivos
```
→ Verifica el tamaño (no exceder límites)
→ Usa formatos correctos (JPG, PNG, MP3, MP4)
→ Prueba con archivo más pequeño
```

### La música no suena
```
→ Los navegadores bloquean autoplay
→ Déjala activada, sonará cuando el usuario interactúe
→ Es normal este comportamiento
```

### Perdí mis datos
```
→ Los datos se guardan en el navegador
→ Si limpiaste caché, se perdieron
→ Exporta regularmente como respaldo
```

## 🆘 Ayuda

### Consola de Debug
```
1. Presiona F12 en el navegador
2. Ve a tab "Console"
3. Revisa si hay errores en rojo
4. Captura pantalla y busca ayuda
```

### Ver Estadísticas
```
1. Presiona F12
2. En Console escribe:
   invitationEditor.getStats()
3. Enter
```

### Limpiar Todo
```
⚠️ CUIDADO: Esto borra TODOS los datos

1. Presiona F12
2. En Console escribe:
   invitationStorage.clearData()
3. Enter
4. Confirma
```

## 📞 FAQ Rápido

**P: ¿Necesito internet para crear la invitación?**
R: Sí, para cargar fuentes y generar QR codes.

**P: ¿Puedo editar después de exportar?**
R: No directamente. Guarda en el editor y exporta de nuevo.

**P: ¿Los invitados necesitan alguna app?**
R: No, solo un navegador web moderno.

**P: ¿Puedo tener múltiples invitaciones?**
R: Una a la vez. Exporta antes de hacer otra.

**P: ¿Se puede imprimir?**
R: Mejor usar el archivo digital. No está optimizado para impresión.

**P: ¿Funciona en celular?**
R: Sí, tanto el editor como las invitaciones son responsive.
